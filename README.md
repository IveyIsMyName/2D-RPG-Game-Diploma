# 2D Action RPG на Unity

Дипломный проект по разработке игрового прототипа в жанре 2D Action RPG на игровом движке Unity.

Проект демонстрирует реализацию основных игровых систем, включая боевую систему, систему характеристик персонажа, дерево навыков, инвентарь, экипировку предметов и систему сохранений. Архитектура проекта построена на модульном подходе с использованием паттерна Finite State Machine (FSM) для управления состояниями персонажа и противников.

## Основные возможности

* Передвижение и прыжки персонажа
* Комбо-атаки и боевая система
* Рывок (Dash) и парирование (Parry)
* Система характеристик персонажа
* Дерево навыков (Skill Tree)
* Инвентарь и экипировка предметов
* Система сохранений и загрузки игры
* Пользовательский интерфейс (UI)
* Искусственный интеллект противников

## Используемые технологии

* Unity
* C#
* Finite State Machine (FSM)
* Scriptable Objects
* JSON Serialization
* Git / GitHub
